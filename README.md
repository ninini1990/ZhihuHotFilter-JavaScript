<h2>使用提示</h2> <p>

**安全性：**<p>
1. 此脚本只发布于，安装说明中所指定的插件商店地址和百度网盘两个分享链接。不要从其它来源下载。<p>
2. 不需要额外输入用户名、密码。此脚本也不会获取、保存、发送你的用户密码。<p>
3. 可检查脚本源码，无任何其它附加操作。<p>

**使用风险提示：**<p>
建议用于随时检查最新一页回答中的被屏蔽情况。<p>
但也请注意不要过于频繁执行，以免被知乎拒绝请求。<p>

**开源协议:**<p>
使用GPL3协议，请保持代码开源及遵守GPL3其他规则。<p>
